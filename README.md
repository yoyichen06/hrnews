# HR NEWS 貼文產生器

社群貼文模板編輯器。手機、電腦都能**快速套版產出貼文**，支援**自製模板**與**分類**。
專為特戰英豪（VALORANT）賽事／遊戲更新新聞這類「固定版型、換文字換圖」的社群貼文設計，
目的是取代「開 Illustrator → 換字 → 拼版 → 匯出」的重複手工流程。

> 純靜態網站（HTML + CSS + 原生 JavaScript），**不需要安裝、不需要建置**，直接丟到 Netlify 就能用。

## 功能

- **內建模板**：特戰英豪新聞（HR NEWS 版）、特戰英豪・快訊、遊戲新聞・聯動、Minecraft 新聞（HR NEWS 版）、技能更新（英雄調整）、地圖輪替（IN/OUT 菱形徽章＋可改地圖名稱）。
- **圖層面板**：像 Illustrator 的圖層清單，可選取、顯示/隱藏、鎖定、上下移、刪除。
- **旋轉**：每個物件都能旋轉（微調面板的「旋轉」）。
- **鍵盤**：Delete／Backspace 刪除；Ctrl+Z 復原、Ctrl+Y 重做；Ctrl+D 複製、Ctrl+C／Ctrl+V 複製貼上；方向鍵微移（Shift 一次 10px）。
- **排列**：水平／垂直置中、複製、圖層順序（上下移／移到最上最下）、圖片水平／垂直翻轉。
- **空心字**：文字可設定「只留外框」的空心效果（例如地圖名稱）。
- **內建元件**：素材庫的「地圖輪替」是元件——點下去會加入菱形徽章＋**可編輯的地圖名稱**（IN／OUT／Map Rotation 是圖、地圖名稱是可改文字）。
  - 「特戰英豪新聞」「Minecraft 新聞」使用你在 Illustrator 做好的 SVG 版型當**固定外框＋兔子 LOGO＋VAL NEWS／MINECRAFT NEWS 標籤**（放在 `templates/*-frame.svg`、內嵌於 `js/frames.js`）；兩張都是**照片全出血**（鋪滿），外框/LOGO 浮在照片上；只留下可換的照片、標題、名稱／版本標籤、主標題上方小圖。
- **匯入 Illustrator SVG 模板**：把你在 AI 做好的版型匯出成 SVG 匯入，SVG 會變成固定版型（外框／LOGO／HR NEWS 都在裡面），系統自動在底下放好背景照片框與上下漸層遮罩、上面放好標題，馬上就能套用。
  - 小技巧：在 Illustrator 用**圖層命名**（例如 `框`/`圖`/`背景`/`漸層`）分開「固定外框」與「可換內容」，之後要拆解會很乾淨。
- **比例／尺寸**：隨時切換 1:1、4:5、9:16、16:9、4:3、3:4。
- **背景進階**：背景照片＋**疊加圖層 Overlay**（材質／光暈）＋**混合模式**（正片疊底／加亮／柔光…）＋**不透明度**。
- **上下漸層遮罩**：每個模板都內建，上緣、下緣各自獨立調整**高度、強度、顏色**；背景照片固定在最底層、滿版。
- **文字外框（描邊）**：每段文字可開/關外框、調整粗細與顏色。
- **陰影**：所有物件都能加陰影，可調角度、距離、模糊、顏色、透明度。
- **復原 / 重做**：Ctrl+Z 復原、Ctrl+Y（或 Ctrl+Shift+Z）重做，記憶 100 步。
- **素材庫（左側面板）**：編輯時左邊的獨立素材庫，可**自訂分類**、上傳／刪除素材；點素材即可套到選取中的圖片欄位（沒選則新增到畫布）。
- **歷史排版**：每次下載會自動存成一份歷史紀錄，之後可從「歷史」打開繼續改或刪除。
- **多頁 / 輪播**：一次可新增很多頁（會複製目前頁，外框／LOGO 都保留，只需改內容），可切換、刪除單頁，支援「下載這頁」或「下載全部」。
- **內建字型 + 上傳字型**：思源黑／宋體、Teko／Oswald／Anton 等電競字型，另含你上傳的 **Foundry Gridnik**、**Minecraft Ten**。
- **預設模板也能改**：所有模板（含預設）都可編輯、儲存（LOGO／外框會被記住）、複製、刪除；誤刪可按「還原預設」找回。
- **標題上方小圖**：主標題上方有可上傳／替換的小圖示 / LOGO 欄位。
- **固定元素**：外框、兔子 LOGO、底部 HR NEWS 預設固定不動；勾選「編輯固定元素」即可解鎖修改／替換。
- **快速填寫**：選好模板後，右側自動列出每個可編輯欄位（主標題、副標題、VAL NEWS 小字＋顏色、LOGO、背景圖…），適合手機快速產圖。
- **微調**：點畫布上的任一物件，即可調整位置、大小、字型、粗細、顏色、對齊、透明度、混合模式、圖層。
- **直接操作**：畫布上可**拖曳移動**、**四角控制點縮放（以物件中心點縮放）**，手機可**雙指縮放**。
- **自製 / 新增 / 刪除模板**：上傳公版加上文字／圖片框／色塊／漸層存成模板；內建模板可「複製為自製」後自由編輯或刪除。
- **分類**：模板依分類篩選；可用「管理分類」自行**新增／改名／刪除**分類（改名會一併更新該分類的模板）。
- **顯示／隱藏元素**：名稱標籤、主標題上方小圖等每個欄位都能勾選要不要（名稱標籤的底框＋文字會一起開關）。
- **外框線**：色塊／標籤底框可開關外框線、調粗細與顏色（例如 MC 版本標籤的方框外框）。
- **邊角**：色塊與圖片框可選「方形／圓角」並自訂圓角值。
- **內建素材**：素材庫附帶內建素材（例如「地圖輪替圖示」，由 `js/builtin-assets.js` 提供），所有裝置都看得到、可直接套用。
- **預覽**：一鍵預覽成品乾淨的樣子（沒有選取框、沒有空框虛線）。
- **下載格式**：可選 **PNG 或 JPG**，一鍵輸出 2× 高解析（預覽 = 下載，完全一致）。
- **跨裝置共用**：用「匯出全部 / 匯入」把自製模板打包成 JSON 檔，在手機↔電腦之間搬移。

## 這次修正／實作了哪些你回報的問題

| 你回報的狀況 | 這裡的作法 |
| --- | --- |
| 可變動文字前面有亂碼（數字） | 全新乾淨的輸入欄位，不會有任何殘留代碼 |
| 中間 VALORANT LOGO 消失 | LOGO 是「圖片框」，可上傳／替換，空的時候只在編輯畫面顯示虛線提示框（**不會被下載進去**） |
| 標題文字沒有置中 | 文字可設定靠左／**置中**／靠右；內建標題預設置中 |
| 副標題的「×」不要更改（代表誰 與 誰） | 「×」設為**固定內容**，不進填寫表單、但仍可自由移動／縮放 |
| overwatch / YOASOBI 之類 LOGO 必須可更改 | 聯動左右兩個 LOGO 都是可替換的圖片框 |
| 改副標題會連主標題一起改 | 每個文字都是**獨立元素**，改一個不會動到另一個 |
| 改技能說明會連技能名稱一起改 | 技能名稱、技能說明各自獨立，互不影響 |
| NERF 字體維持一致（預覽/下載跑掉） | 預覽與下載走**同一段 Canvas 繪製**、繪製前先確保字型載入 → 兩者 100% 一致 |
| 圖片要以物件中心點放大（目前從左上角） | 縮放一律**以元素中心為基準**，中心不動 |
| 「×」也要能移動、調整大小 | 所有元素（含固定文字）都能拖曳、縮放 |
| 技能說明要能改位置 | 可用拖曳或 X／Y 滑桿調整位置 |
| 左側 LOGO 可替換成圖片、可中心放大、滑桿移動 | 圖片框皆支援替換、中心縮放、X／Y 滑桿 |
| Minecraft 背景更改圖片沒反應 | 背景是可替換的圖片元素，換圖即時生效，且可移動／中心縮放 |

## 使用方式

### A. 用內建模板產一張貼文
1. 在首頁點一張模板卡。
2. 右側「快速填寫」把文字打上、LOGO／背景圖上傳。
3. 需要微調位置或大小時，點畫布上的物件，切到「微調」。
4. 按右上角「下載 PNG」。

### B. 把自己的公版變成可編輯模板
1. 首頁點「＋ 自製模板」。
2. 填模板名稱、分類，選尺寸（方形 / 直式 / 限動）。
3. 「上傳背景圖」放入你的公版；再用「＋ 文字 / ＋ 圖片框 / ＋ 色塊」擺上可變動的欄位。
4. 拖曳／縮放調好版面，按「儲存模板」。之後它就會出現在對應分類，可重複套用。

### C. 匯入你在 Illustrator 做好的 SVG 版型
1. 在 Illustrator 把版型（外框、兔子 LOGO、HR NEWS 帳號等固定元素）排好。
2. **建議把文字先「建立外框」(Create Outlines)**，這樣匯出後字型一定不會跑掉；需要每篇換字的地方，留給這裡的文字欄位來擺。
3. 匯出成 SVG，回到首頁點「匯入 SVG 模板」選檔。
4. 匯入後會自動進入編輯：SVG 是固定版型，底下已放好背景照片框與上下漸層遮罩、上面有主標題／副標題。調好位置按「儲存模板」。

### D. 手機 ↔ 電腦共用模板
- 在 A 裝置按右上「匯出全部」下載 JSON。
- 到 B 裝置按「匯入」選那個 JSON 檔即可。
- （自製模板存在瀏覽器 localStorage，清除瀏覽器資料會消失，記得偶爾匯出備份。）

## 雲端同步（Supabase，跨裝置）

用 email 登入，把**模板、素材庫、歷史排版**同步到雲端，換手機或電腦都能拿到同一份資料。

1. 到 [supabase.com](https://supabase.com) 開一個免費專案。
2. 專案的 **SQL Editor** 執行以下建表語法（一次就好）：

   ```sql
   create table if not exists items (
     uid uuid not null default auth.uid(),
     kind text not null,
     item_id text not null,
     data jsonb,
     deleted boolean default false,
     updated_at timestamptz default now(),
     primary key (uid, kind, item_id)
   );
   alter table items enable row level security;
   create policy "own" on items for all
     using (uid = auth.uid()) with check (uid = auth.uid());
   ```
3. **Project Settings → API** 複製 `Project URL` 與 `anon public key`。
4. 回到網站點右上「☁ 同步」→ 貼上 URL 與 key →「儲存連線設定」→ 用 email 註冊 / 登入。
5. 登入後會自動同步；之後在任何裝置用同一組帳號登入就會抓到同一份資料。

> anon public key 放在前端是安全的（受 Row Level Security 保護，每個帳號只讀寫自己的資料）。
> 若在 Supabase 開了「Confirm email」，註冊後要先去收信驗證再登入。

## 部署到 Netlify

這是純靜態站，沒有建置步驟：

- **拖拉部署**：把整個資料夾拖到 Netlify 的 Deploys 頁面即可。
- **Git 連動**：連結此 repo，設定 `publish = "."`、build command 留空（`netlify.toml` 已設定好）。

本機預覽（因為用到 ES modules，需要透過伺服器開，不能直接雙擊 `index.html`）：

```bash
npx http-server -p 8123
# 開 http://127.0.0.1:8123
```

## 建議的 AI 工作流

這個工具負責「排版與產圖」，AI 可以負責「內容」：

1. **蒐集／整理**：把官方公告、patch note 丟給 AI，請它濃縮成主標題、副標題、重點條列。
2. **翻譯／潤稿**：AI 幫忙翻成中文、統一語氣、算好字數避免爆版。
3. **套版**：把 AI 產出的文字貼進對應模板的欄位，換上 LOGO／截圖，下載。
4. **一稿多平台**：同一份內容，用不同尺寸（方形 / 限動）各出一張。

如此就能把原本「找資料 → 開 Illustrator → 換字拼版 → 發文」壓縮成「AI 整理文字 → 套版下載 → 發文」。
